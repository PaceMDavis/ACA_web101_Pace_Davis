Beam Me Up Version Control
August 30, 2019
GIT is a version control system used to manage projects across the world. It allows projects to be managed locally and centrally at the same time. This means that multiple can access and store entire projects on their computers, work on them, and then commit changes. Projects can also be broken into branches, which can be edited and uploaded to merge back into one, if more than one person is working on similar sections.
